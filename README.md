# 🎨 Vibe Coding — Experimental Website

Eine interaktive Playground-Website mit Fokus auf kreative Interaktionen und modernes Design.

## 🚀 Features

### Interaktive Elemente
- **Custom Cursor**: Benutzerdefinierter Cursor mit Follower-Effekt (nur Desktop)
- **Hover-Effekte**: 3D-Transformationen auf Grid-Items mit Parallax-Feel
- **Scroll-Animationen**: Elemente faden sanft ein beim Scrollen
- **Text Reveal**: Wort-für-Wort Animation im Text-Bereich
- **Card Interactions**: Bounce-Effekt beim Klicken auf Project Cards

### Design
- **Mutige Farben**: Inspiriert von deinen Bildern (Rot, Orange, Pink)
- **Fließende Shapes**: Animierte Hintergrund-Elemente
- **Moderne Typografie**: Große, kraftvolle Headlines
- **Responsive**: Funktioniert auf Desktop und Mobile

## 📁 Struktur

```
vibecoding_website/
├── index.html      # HTML-Struktur mit allen Sektionen
├── styles.css      # Komplettes Styling mit CSS-Variablen
└── script.js       # Alle interaktiven Features
```

## 🎯 Verwendung

1. Öffne `index.html` in deinem Browser
2. Bewege die Maus über verschiedene Elemente
3. Scrolle durch die Seite für Animationen
4. Klicke auf Grid-Items und Cards für Interaktionen

## 🛠️ Anpassungen & Experimente

### Farben ändern
In `styles.css` die CSS-Variablen anpassen:
```css
:root {
    --color-primary: #FF3366;    /* Hauptfarbe */
    --color-secondary: #FF6B35;  /* Zweite Farbe */
    --color-accent: #F7931E;     /* Akzentfarbe */
}
```

### Neue Interaktionen hinzufügen
In `script.js` kannst du:
- Event Listener für neue Elemente erstellen
- Animationen anpassen (Geschwindigkeit, Easing)
- Neue Hover-Effekte programmieren
- Scroll-getriggerte Effekte erweitern

### Weitere Sektionen
In `index.html`:
- Kopiere eine `<section>` und passe sie an
- Füge neue Grid-Items oder Cards hinzu
- Experimentiere mit verschiedenen Layouts
